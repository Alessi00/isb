Cancer Gateway in the Cloud

Access, Explore and Analyze Large-Scale Cancer Data Through the Google Cloud with ISB-CGC

Data Browsers

BigQuery Table Search

Browse a selection of metadata and molecular cancer data from the Genomic Data Commons and other sources. These tables are hosted on Google BigQuery, allowing for easy and cheap computation and retrieval with the use of Structured Query Language (SQL).

Data Browser

Explore a comprehensive selection of cancer related data files hosted in Google Cloud Storage Buckets, such as raw sequencing, cancer nucleotide variation, pathology or radiology images, etc through a user-friendly file browser

Chromosomal Aberrations & Gene Fusions DB

The Mitelman Database of Chromosome Aberrations and Gene Fusions in Cancer relates cytogenetic changes and their genomic consequences, in particular gene fusions, to tumor characteristics. Browse this data using the Mitelman Database UI.
Learn
Launch

Resources

Interactive Patient Cohort Exploration

Interactive web interface to build and explore cohorts based on clinical demographics (age, race, gender, tumor type, etc) and molecular filters (e.g. gene mutations). Compare patient cohorts with various interactive exploration tools including IGV viewer, image viewers, and analytical visualization features.
Learn
Launch

Programmatic Access

Access and explore data through command-line interfaces including: Python, R, Google Compute Engines (VMs), local terminals. The ISB-CGC API can be used to programmatically interact with cohorts as well as access data about saved cohorts, metadata about patients & samples, and info about your Google Cloud Project.
Learn
Launch

Notebooks

A collection of notebooks written in R and python, compiled by the ISB-CGC team to serve as both tutorials or analysis tools for both novice and experienced users. Includes a set of notebooks reproducing the high level functionality of Regulome Explorer.
